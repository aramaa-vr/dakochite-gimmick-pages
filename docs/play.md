# 遊び方（操作）

VRChat上での操作（エクスプレッションメニュー／掴み方） 

---

## 1. エクスプレッションメニュー

![エクスプレッションメニュー]({{ "/assets/img/dakochite/menu-expression.webp" | relative_url }})
![エクスプレッションメニュー（詳細）]({{ "/assets/img/dakochite/menu-expression-sub.webp" | relative_url }})

### 機能一覧（まずは概要だけ）

| 機能 | 概要（要点） |
|---|---|
| だこちてモード | みんなにGrabで抱っこされる／抱っこ中にUseで固定 |
| 飛行モード | 自分をGrabで掴んで飛行／Useで固定（地面に隠れたり空中固定も可能） |
| 狂気のバトルモード | ギミック導入者同士でバトル（先につかんだ方が勝ち） |
| アピール | アピールのエフェクト量を0%〜100%で調整 |
| 説明書 | VRChat内で表示できる簡易説明書 |
| だこちて視点On | 抱っこされた時に視点移動（Quest版は利用不可） |
| 音とエフェクト On | 音とパーティクルが出る（On/Off） |
| ビューポイントNear | 視点に髪の毛や鼻が映る時に調整（上げすぎ注意） |
| デバック | Onにするとつかまれた状態を再現（説明書では基本放置推奨） |

### 各項目の補足（説明書の表より）

#### だこちてモード

- Grabで抱っこされる機能。
- 抱っこ中にUseボタンで固定。
- 固定したまま放置された場合は、「だこちてモード」をOFFで元に戻ります。

#### 飛行モード

- 自分をGrabで掴んで飛行。
- Useボタンで固定でき、地面に隠れたり空中固定も可能です。
- 飛行して他人に迷惑をかけないでね。

#### 狂気のバトルモード

- ギミック導入者同士でバトル。
- 先につかんだ方が勝ち、負けた方は空中にぶん投げられる。
- 倒れた場合は、両手をグーのジェスチャー、または「バトルモード」をOffで元に戻ります。

#### アピール

- アピールのエフェクトが出る。
- 0%〜100%で量が増減します。

#### 説明書

- VRChat内で表示できる簡易説明書です。

![VRChat内説明書]({{ "/assets/img/dakochite/menu-help.webp" | relative_url }})

#### だこちて視点On

- 抱っこされた時、視点移動する。
- Quest版は利用不可です。

#### 音とエフェクト On

- 音とパーティクルが出る（On/Off）。

#### ビューポイントNear

- だこちて視点で髪の毛や鼻が画面に映る場合に使用。
- パーセンテージを上げると鼻先や髪の毛のオブジェクトが消えます。
- 上げすぎると相手の顔まで消える可能性があるため、視界をふさぐものがギリギリ見えない数値に設定してください。

#### デバック（説明書表記）

- Onにするとつかまれた状態を再現。
- On/Offどっちになってても基本放置してください。

## 3. 固定（ピン留め）

だこちてモード／飛行モードいずれも、 **Useボタンで固定** できます。  
固定したまま放置された場合は「だこちてモードをOFF」で元に戻ります。
