# 遊び方（操作）

本文は同梱説明書（HTML）「4, 遊び方」を元に整理しています。

## 1. エクスプレッションメニュー

![エクスプレッションメニュー]({{ site.baseurl }}/assets/img/dakochite/menu-expression.webp)
![エクスプレッションメニュー（詳細）]({{ site.baseurl }}/assets/img/dakochite/menu-expression-sub.webp)

### 機能一覧（まずは概要だけ）

| 機能 | 概要（要点） |
|---|---|
| だこちてモード | みんなにGrabで抱っこされる／抱っこ中にUseで固定 |
| 飛行モード | 自分をGrabで掴んで飛行／Useで固定（地面に隠れたり空中固定も可能） |
| 狂気のバトルモード | ギミック導入者同士でバトル（先につかんだ方が勝ち） |
| アピール | アピールのエフェクト量を0%〜100%で調整 |
| 説明書 | VRChat内で表示できる簡易説明書 |
| だこちて視点On | 抱っこされた時に視点移動（Quest版は利用不可） |
| 音とエフェクト On | 音とパーティクルが出る（On/Off） |
| ビューポイントNear | 視点に髪の毛や鼻が映る時に調整（上げすぎ注意） |
| デバック | Onにするとつかまれた状態を再現（説明書では基本放置推奨） |

### 各項目の補足（説明書の表より）

#### だこちてモード

- Grabで抱っこされる機能。
- 抱っこ中にUseボタンで固定。
- 固定したまま放置された場合は、「だこちてモード」をOFFで元に戻す、という記載があります。

#### 飛行モード

- 自分をGrabで掴んで飛行。
- Useボタンで固定でき、地面に隠れたり空中固定も可能、と記載があります。
- 説明書には「飛行して他人に迷惑をかけないでね」と注意があります。

#### 狂気のバトルモード

- ギミック導入者同士でバトル。
- 先につかんだ方が勝ち、負けた方は空中にぶん投げられる、と記載があります。
- 倒れた場合は、両手をグーのジェスチャー、または「バトルモード」をOff、と記載があります。

#### アピール

- アピールのエフェクトが出る。
- 0%〜100%で量が増減、と記載があります。

#### 説明書

- VRChat内で表示できる簡易説明書、と記載があります。

![VRChat内説明書]({{ site.baseurl }}/assets/img/dakochite/menu-help.webp)

#### だこちて視点On

- 抱っこされた時、視点移動する。
- Quest版は利用不可、と記載があります。

#### 音とエフェクト On

- 音とパーティクルが出る（On/Off）。

#### ビューポイントNear

- だこちて視点で髪の毛や鼻が画面に映る場合に使用。
- パーセンテージを上げると鼻先や髪の毛のオブジェクトが消える、と記載があります。
- 上げすぎると相手の顔まで消える可能性があるため、視界をふさぐものがギリギリ見えない数値にするよう案内されています。

#### デバック（説明書表記）

- Onにするとつかまれた状態を再現。
- On/Offどっちになってても基本放置、と記載があります。

## 3. 固定（ピン留め）

だこちてモード／飛行モードいずれも、説明書では **Useボタンで固定** と書かれています。  
固定したまま放置された場合は「だこちてモードをOFFで元に戻る」との説明があります。
